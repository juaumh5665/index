🖨️ Impressão silenciosa do StackBar

Configurar Chrome com --kiosk-printing pra cupons saírem direto na impressora térmica sem mostrar diálogo
Por que isso é necessário: Por padrão, o Chrome abre uma janela de "Imprimir" a cada cupom — pediria pra você clicar "Imprimir" toda vez. O modo --kiosk-printing faz o Chrome imprimir direto na impressora padrão, sem nenhum clique. Essencial pro StackBar funcionar 100% automático.
1
Configurar impressora padrão do Windows

Antes de qualquer coisa, o Windows precisa saber qual impressora usar quando o Chrome mandar imprimir "sem perguntar".

Passos:

  1. Abre o menu Iniciar → Configurações → Dispositivos → Impressoras e scanners
  2. Desmarque "Permitir que o Windows gerencie minha impressora padrão" (essa opção troca a padrão automaticamente, atrapalha)
  3. Clique na sua impressora térmica (ex: EPSON TM-T20, Bematech MP-4200) → Gerenciar → Definir como padrão
✓ Como confirmar: abra qualquer PDF e dê Ctrl+P. A impressora térmica deve aparecer pré-selecionada.
2
Criar atalho do Chrome com a flag

O --kiosk-printing precisa ser passado quando o Chrome inicia. Vamos criar um atalho dedicado.

  1. Clique com botão direito na área de trabalho → Novo → Atalho
  2. Cole o caminho:
"C:\Program Files\Google\Chrome\Application\chrome.exe" --kiosk-printing --app=https://stackos.com.br/app
  1. Nome do atalho: StackOS Bar (ou o que preferir)
  2. Concluir
✓ Dica: use --app=URL no lugar de só --kiosk-printing sozinho. Isso abre o StackOS em modo "app" (sem barra de endereço, sem abas), parecendo um software dedicado.

No Mac é mais chato porque não tem como criar atalho gráfico facilmente. Use o terminal:

/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --kiosk-printing \
  --app=https://stackos.com.br/app

Pra criar um atalho clicável, abra o Automator → "Aplicativo" → "Executar Script Shell" → cole o comando acima → salve como app na área de trabalho.

Cria um arquivo ~/.local/share/applications/stackos-bar.desktop:

[Desktop Entry]
Name=StackOS Bar
Exec=google-chrome --kiosk-printing --app=https://stackos.com.br/app
Terminal=false
Type=Application
Icon=google-chrome
3
Testar a impressão silenciosa
  1. Abra o atalho que você criou (StackOS Bar)
  2. Faça login normalmente
  3. Vá em Convites → Config (do StackBar) e marque "Esta máquina recebe impressões"
  4. No StackBar PWA do celular, registre um item de teste numa comanda
  5. O cupom deve imprimir sem nenhum diálogo aparecer na tela do PC
✓ Sucesso: Cupom saiu na térmica sem o admin nem perceber. ✨
✗ Falhou: Apareceu diálogo "Imprimir"? Significa que a flag não foi aplicada. Confira:
  • Você fechou TODAS as janelas do Chrome antes de abrir pelo atalho? (a flag não funciona se já tem Chrome aberto)
  • O caminho do Chrome no atalho está correto? (testa abrir o atalho — se abrir, o caminho tá ok)
  • A impressora padrão do Windows é mesmo a térmica?
4
Modo Solo (clube sem máquina dedicada do bar)

Se o seu clube tem apenas 1 computador (admin no caixa + impressora compartilhada), ative o "Modo Solo" no Painel do StackBar:

  1. Acesse o painel de configuração do StackBar (geralmente em stackos.com.br/stackbar-painel.html)
  2. Marque a opção "Permitir admin imprimir (modo solo)"
  3. Salvar

A partir daí, mesmo o admin logado vai receber os cupons automaticamente. A impressão é via iframe oculto — não interrompe o trabalho do admin no caixa.

Checklist de configuração
  • Impressora térmica conectada e instalada no Windows
  • Impressora térmica definida como padrão do Windows
  • "Permitir que o Windows gerencie impressora padrão" desmarcado
  • Atalho do Chrome com --kiosk-printing criado
  • StackBar marcado como "esta máquina recebe impressões" no PWA
  • Auto-impressão ligada no Painel do StackBar (config geral)
  • "Modo Solo" ativo no Painel (se for o caso de admin imprimir)
  • Teste de cupom funcionou sem mostrar diálogo
